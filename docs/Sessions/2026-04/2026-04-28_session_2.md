## What was decided

- **0.855 in-sample Sharpe is pre-governance, not reproducible**: The 0.855 was produced on the
  run that GENERATED the lifecycle pause decisions for atr_breakout and momentum_edge. Both edges
  were at full weight (2.5 and 1.5) during that run and contributed +$4,263 combined. After
  lifecycle correctly quarantines them, subsequent in-sample runs show ~0.256 Sharpe. The 0.855
  is a valid historical measurement of Phase 2.10 on the 2021-2024 window; it is not a stable
  operational baseline.

- **Operational baseline is 0.256 Sharpe** (neutral governor + PAUSED_MAX_WEIGHT cap, run
  e5055f4e). SPY 2021-2024 Sharpe: 0.875. Gap: **-0.619 Sharpe**. This is larger than the
  pre-Phase 2.10 gap (0.875 - 0.403 = 0.472) because Phase 2.10 macro edges barely fire
  (macro_credit_spread: 0 trades, macro_yield_curve: -$784 loss) and the soft-pause weight
  on atr_breakout now has a 0.5 cap instead of 0.625.

- **Governor should reset to neutral (1.0) for in-sample backtests**: The governor's persistent
  learned weights are production state, not measurement state. Pre-training on OOS data and then
  using those weights in the in-sample window injects forward-looking signal. Same principle as
  `lifecycle_readonly`: backtesting-as-measurement needs clean initial conditions. For now, reset
  manually (`edge_weights.json` all-1.0). Should be automated with a `--reset-governor` flag.

## What was learned

- **The 0.855 measurement artifact**: A lifecycle run that GENERATES pause decisions naturally
  includes those edges at full weight during the run. Subsequent runs retroactively apply soft-pause
  to the entire historical window. This makes the measurement unreproducible after the lifecycle
  fires. The correct way to compare Phase 2.10 impact: measure both WITH and WITHOUT the Phase 2.10
  edges under the same lifecycle conditions (both at soft-pause for atr_breakout/momentum_edge).

- **Governor affinity is a measurement contaminant**: If the governor learned from adversarial OOS
  data, those weights hurt profitable edges in the in-sample window. The -0.103 Sharpe effect is
  not noise — it's systematic suppression of atr_breakout (governor weight 0.587) and momentum_edge
  (0.528) learned from the 2025 market (tariff spike, high vol). Always reset to neutral for clean
  measurement.

- **Phase 2.10 macro edges are silent in the in-sample window**: macro_credit_spread, pead_v1,
  most macro edges fire 0-3 times in 2021-2024. Either their signal thresholds are too conservative
  or the 2021-2024 macro environment doesn't trigger them. This needs investigation before the
  discovery loop can be evaluated.

- **"Unknown" exit losses are a major drag (-$14,832 from 885 exits)**: These are stop-loss and
  take-profit fills with no edge attribution. The volume scales with position count (more atr_breakout
  entries → more stops → more Unknown exits). This represents real loss but the magnitude (-$16.8
  per exit on average) is suspicious — may indicate a stop placement or sizing regression relative
  to the 0.855 run where Unknown was only -$3,681.
